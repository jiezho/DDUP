# DDUP 数据库文档

## 1. 总览

DDUP 使用 PostgreSQL 作为关系数据库。当前阶段的数据结构以 SQLAlchemy 模型定义为准。

## 2. 自动生成 Schema（保持后续更新可同步）

数据库表结构文档由脚本从 SQLAlchemy 的 `Base.metadata` 生成：
- 生成脚本：`apps/api/tools/generate_db_schema_md.py`
- 生成产物：`docs/generated/db_schema.md`

当你新增/修改模型（例如新增表、字段、索引）后：
1. 运行后端测试
2. 运行生成脚本刷新 `docs/generated/db_schema.md`
3. 提交到 git，确保文档与代码同步
